inkOS is a minimal text launcher

optimized for e-ink devices and keypad/qwerty phones

What is the point of inkOS?

inkOS is different from other minimalist android launchers because it's optimized for e-ink, keypad and qwerty phones. Almost all of the interface is operable with D-pad, so touch input is optional. It removes all animations and replaces smooth scrolling with discrete, paged navigation, making it comfortable for users who are sensitive to motion and ideal for slow-refresh e-ink displays. To reduce distraction and app hopping, inkOS surfaces notifications directly on the launcher, letting you stay informed without constantly opening apps.

Core Features

Customizable Home Screen

Set the number of home apps, pages, and easily drag & drop to rearrange. Auto-divide apps across multiple pages with visual indicators.

Flexible Typography

Universal font support with individual customization for apps, clock, battery, and notifications. Adjustable text sizes for perfect readability.

Theme & Visual Customization

Light/dark themes with custom element colors. E-ink optimization with auto-refresh. Show/hide status bar and UI elements as needed.

Smart Gestures

Volume key navigation, customizable double-tap actions, swipe gestures for quick access to notifications and apps.
